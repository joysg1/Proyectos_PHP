# Sistema de Análisis de Producción Celular por Actividad Física

## Descripción General

Sistema completo de análisis de datos que estudia la relación entre la actividad física diaria y la producción de células en el organismo. El sistema combina tecnologías modernas de backend (Python) y frontend (PHP) para ofrecer un análisis robusto y visualmente atractivo.

## Características Principales

### Backend (Python)
- **API RESTful** con Flask
- **Procesamiento de datos** con Pandas y NumPy
- **Machine Learning** con Scikit-learn (Random Forest)
- **Visualizaciones** con Seaborn, Matplotlib y Plotly
- **Base de datos** JSON robusta con 1000+ registros

### Frontend (PHP)
- **Interfaz moderna** con diseño oscuro responsive
- **Gestión de datos** (CRUD completo)
- **Visualizaciones interactivas** en pestañas separadas
- **Predicciones en tiempo real** usando el modelo ML

### Gráficos Implementados
1. **Área sobre la curva** - Producción acumulada en el tiempo
2. **Gráfico Radar** - Comparación multidimensional por actividad
3. **Barras Apiladas** - Producción por actividad y género
4. **Gráfico de Pastel** - Distribución porcentual por actividad
5. **Mapa de Calor** - Correlaciones entre variables

### Como ejecutar
### cd backend ---> python app.py
### cd frontend ---> php -S localhost:8000 ---> abrir navegador web de preferencia ---> http://localhost:8000/index.php
