# Sistema de Análisis de Crecimiento Poblacional en América Latina

Este proyecto proporciona una plataforma integral para analizar y visualizar datos de crecimiento poblacional en países de América Latina. 

## Características principales:
- Dashboard interactivo con métricas clave
- Visualizaciones avanzadas de datos poblacionales
- Análisis predictivo usando machine learning
- Interfaz moderna y responsive

Ideal para investigadores, analistas de datos y cualquier persona interesada en entender las tendencias demográficas de la región.

## Ejecutar backend:
cd backend ---> python app.py

## Ejecutar frontend:
cd frontend ---> php -S localhost:8000 ---> http://localhost:8000/index.php
