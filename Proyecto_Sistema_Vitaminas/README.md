# Sistema de Análisis de Vitaminas y Glóbulos Rojos

Sistema completo para analizar el impacto de diferentes vitaminas en la producción de glóbulos rojos mediante visualizaciones avanzadas y machine learning.

## Características

- **Backend Python**: API REST con Flask para procesamiento de datos
- **Frontend PHP**: Interfaz web moderna con tema oscuro
- **Visualizaciones**: Gráficos de área, radar, barras apiladas y pastel
- **Machine Learning**: Modelo de Random Forest para predicciones
- **Tema Oscuro**: Interfaz profesional y moderna
- **Arquitectura Escalable**: Estructura de archivos organizada
